Мой проект, это сайт для обсуждения тем аниме тематики, а так же просмотр новостей связанные с аниме

Функции сайта:
  1) Регистрация/Авторизация
  2) Редактирование профиля
  3) Написание авторизированным пользователям форумов
  4) Добавление новостей через аккаунт администратора

Стек данного проекта:
  1) Сервер работает на движке Django
  2) Для работы с базой данных используется SQLite3
  3) Используется шаблонизатор Jinja
  4) 

Установка:
  1) Клонируем проект
  2) Создаём виртуальное окружение / активируем его
  3) Выбираем интерпритатор активированного окружения
  4) Устанавливаем библиотеки через requirements.txt
  5) Осуществялем миграции, после, запускаем через консоль setup.py два раза
